# Roadmap

本 Roadmap 用于说明公开方向，不承诺具体发布日期。

## 当前主线

- 简历模板与编辑
- 目标岗位分析
- 岗位竞争力诊断
- 岗位投递文案
- PDF 导出
- 移动端基础编辑与预览

## 近期优化

- 更多行业模板和行业示例
- 移动端长文本编辑体验优化
- PDF 导出前多页预览一致性继续打磨
- 投递文案继续强化证据解释
- 更清楚的隐私说明和 AI 使用边界
- 用户反馈入口和额度申请流程

## 暂不做

- 自动投递
- 批量爬取招聘网站
- 自动伪造经历
- 承诺录用概率
- 向前端暴露模型 API Key
- 开放任意 URL 转 PDF
