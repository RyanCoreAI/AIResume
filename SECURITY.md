# Security Policy

## Reporting a Vulnerability

请不要在公开 Issue 中发布可复现攻击步骤、真实用户数据、简历内容、JD 原文、日志、密钥或访问 token。

如果你发现安全问题，请通过 [yanxai.com](https://yanxai.com) 的反馈入口联系，并尽量只描述：

- 影响范围
- 受影响页面或功能
- 大致复现条件
- 是否涉及真实用户数据

在确认安全边界前，请不要公开传播漏洞细节。

## Scope

当前公开仓库只包含产品展示文档和截图，不包含生产源码、配置或密钥。

线上产品相关安全问题包括但不限于：

- 账号登录与会话
- 简历、版本、分享链接、PDF 导出权限
- XSS / CSRF / CORS
- 文件上传
- PDF 渲染
- AI 输出注入
- 越权访问
